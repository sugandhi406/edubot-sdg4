# 📚 EduBot — AI Study Assistant

A chatbot that helps students study better. Built for **SDG 4: Quality Education** as part of Lenovo LEAP 2026 Internship.

---

## Why I Made This

Students always have doubts — but teachers aren't always available, tuition is expensive, and googling gives you 10 different confusing answers. EduBot is basically a study buddy that's available 24/7 and actually gives useful answers.

---

## What Can It Help With?

- Study tips (Active Recall, Pomodoro, Spaced Repetition)
- Exam preparation — what to do 1 week, 3 days, 1 day before
- Motivation when you feel like giving up
- Managing exam stress
- Time management and making a study schedule
- Free learning resources (Khan Academy, NPTEL, GeeksforGeeks)
- CS and DSA help for engineering students
- Staying focused and avoiding distractions

---

## How to Run It

1. Download the HTML file
2. Open it in any browser
3. Start typing!

No setup. No installation. No internet needed after it loads.

---

## Tech Used

- HTML, CSS, JavaScript — all in one single file
- Rule-based keyword matching for AI responses
- CSS animations for the typing effect and smooth UI
